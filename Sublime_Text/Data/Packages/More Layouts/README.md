More Layouts
================

## Summary
Add more layouts 

## Screenshots
![ScreenShot](https://raw.github.com/unknownuser88/morelayouts/master/1.gif)

## Key Binding

The default key binding is "alt+shift+6" and "alt+shift+7" and "alt+shift+0".
